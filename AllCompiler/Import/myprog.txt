Т.к. необходимо вводить типы, то синтаксис нужно менять

Какие есть типы:
1. Обычные числа (int)
2. Указатели (& + type)
3. Строки (str)
    - Строка просто представляет собой массив из ASCII кодов
    - Вывод будет производиться посимвольно
    - Походу конец будет обозначаться так же ('\0')

Поддерживаются массивы и структуры
Массивы
    - Только int и только одномерные (пока)
    - int mass<10> = @1, 3, 5, 2, 5, 2, 1, 5@
    - Обращение по индексу mass<4> = 4
    - Проверка на индекс пока неизвестно, будет или нет


Структуры
    - Из разных типов
    - block NameStruct <    |                                                 |   block NameStruct <
            int a           |   Или можно использовать синтаксис и написать   |       int a ~ int b ~ str name<10>
            int b           |                                                 |   >
            str name<10>    |                                                 |
      >                     |                                                 |

Для обычных переменных:
    al i32 num = 0
    al i32 num

Указатели:
    al &i32 num = &num
    al &i32 num

Объявление функции:
    ser NameFunction<int arg, int* arg, str string> -> int (int*, str)
